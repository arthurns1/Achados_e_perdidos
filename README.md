# 🧭 Achados e Perdidos

Sistema web de **Achados e Perdidos** desenvolvido para facilitar o registro, a busca e a recuperação de itens perdidos, promovendo facilitar a recuperação de itens perdidos no campus.

O projeto é dividido em **Front-end** (React) e **Back-end** (Node.js com TypeScript).

---

## 🚀 Tecnologias Utilizadas

### 🔹 Front-end
- React
- JavaScript
- HTML5
- CSS3
- React Router DOM

### 🔹 Back-end
- Node.js
- TypeScript
- Express
- Banco de Dados (PostgreSQL)
- JWT (Autenticação)
- Multer (Upload de imagens)
- Cors

---

## 📌 Funcionalidades

- Cadastro de usuários
- Autenticação e autorização
- Cadastro de itens perdidos
- Upload de imagens dos itens
- Registrar a devolução de itens
- Visualização de detalhes do item guardado
- Interface responsiva
